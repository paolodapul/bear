## Bear.js

Bear is an opinionated Node.js starter template.

Features:

- Linting
- Formatting
- TypeScript-ready
- Compiled with SWC

Upcoming:

- [ ] Validation with Zod
- [ ] Routes
- [ ] GitHub Actions integration
- [ ] Production monitoring with PM2
- [ ] Prisma integration
- [ ] Code-first GraphQL schema definition with TypeGraphQL
- [ ] Docker
- [ ] Use Postgres as database
- [ ] Authentication
- [ ] Testing with Jest
